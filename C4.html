변수 선언

 변수를 선언하는 방법은 두 가지로 나뉜다.
    정수형 변수 : 정수를 저장하는 변수로 char, short, long, int가 있다.
    실수형 변수 : 실수를 저장하는 변수로 float, double, long double이 있다.

사용 예시

#include


int main (void)

{

int a; ------------- 정수형 변수 선언

float b; ----------- 실수형 변수 선언


printf("%d\n",a); ----------- 변수 a에 저장된 값을 출력

printf("%f\n",b); ----------- 변수 b에 저장된 값을 출력


return 0;

}

변수 이름을 지을 시 주의할 점

  • 특수 기호, 공백 문자, 변수 이름 맨 처음에 숫자를 사용하면 안 된다.
  • C언어에서 사용되는 키워드를 변수 이름에 사용하면 안 된다.
  • C언어는 대소문자를 구분하므로 변수의 이름에 유의해서 지어야 한다.
